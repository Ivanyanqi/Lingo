## [v0.3.0] — Bug 修复与体验打磨

> 本版本专注于修复 v0.2.0 中已知的所有问题，全面提升稳定性。

### 修复

- **引擎切换立即生效** — 此前在设置中切换翻译引擎需要重启才能生效。现在 `TranslationViewModel` 持有响应式的 `currentEngine` 属性，切换时立即替换底层 service 实例并清空缓存，不会再出现旧引擎的结果残留。
- **收藏按钮精确匹配当前条目** — 翻译结果区的收藏星标此前使用 `entries.first` 获取条目，在列表并发更新时可能收藏到错误的记录。现在通过 `currentEntryID`（`HistoryStore.add()` 返回的 `UUID`）精确查找对应条目。
- **500 字符上限真正生效** — 此前界面上显示了字符计数但从未实际截断输入。现在在 `inputText.didSet` 中强制截断，字符数达到 450 时计数器变红提示。

### 新增

- **划词悬浮按钮开关** — 设置页「通用」区域新增开关，可随时启用或关闭选中文字后出现的「译」悬浮按钮，偏好持久化保存。`SelectionButtonController` 改为单例。
- **网络状态感知** — 新增 `NetworkMonitor`（基于 `NWPathMonitor`）实时检测网络连通性。断网时翻译请求立即返回明确错误，而不是等待 10 秒超时。翻译页顶部会显示橙色提示条（已缓存的翻译仍可正常使用）。

### 变更

- **系统 TTS 升为主要朗读引擎** — `SpeechService` 此前优先调用非官方 Google TTS 接口，存在被封禁或限速的风险。现在优先检查 `AVSpeechSynthesisVoice` 是否支持目标语言，支持则直接使用系统合成器（离线可用）。Google TTS 保留为降级备用，网络失败时最终降级到英语系统声音。

## [v0.2.0] — 全面功能迭代

> 本版本将 MVP 升级为完整的日常使用工具，新增历史记录、多引擎支持和更智能的交互体验。

### 新增

- **翻译历史记录** — 每次翻译自动保存到本地（最多 50 条）。历史 Tab 展示原文、译文、语言对和相对时间。支持单条收藏（⭐）、删除，以及全量导出为 CSV 文件。
- **多语言支持** — 在中英互译基础上新增 8 种目标语言：日语 🇯🇵、韩语 🇰🇷、法语 🇫🇷、西班牙语 🇪🇸、德语 🇩🇪、葡萄牙语 🇧🇷、俄语 🇷🇺。语言选择持久化保存。
- **DeepL 引擎** — 接入 DeepL API（免费版和付费版均支持），需在设置中填写 API Key。
- **OpenAI 引擎** — 接入 OpenAI `gpt-4o-mini`，提供上下文感知的高质量翻译，需填写 OpenAI API Key。
- **设置页引擎切换** — 可在 MyMemory（免费，无需 Key）、DeepL、OpenAI 三种引擎间切换，API Key 存储在 `UserDefaults`。
- **LRU 翻译缓存** — 线程安全的 100 条 LRU 缓存（`TranslationCache`），相同文本不重复请求 API。
- **输入防抖** — 面板输入停止 400ms 后才触发翻译，减少不必要的 API 调用。
- **开机自启动** — 设置页「通用」区域新增开关，基于 `SMAppService` 实现登录时自动启动。
- **划词悬浮翻译按钮** — 在任意应用中选中文字后，光标附近会出现「译」小按钮，点击即可翻译并弹出悬浮结果窗。基于 `NSEvent` 全局监听 + Accessibility API 实现。
- **菜单栏预览** — 翻译完成后，结果的前 12 个字符会短暂显示在菜单栏图标旁。
- **退出按钮** — 设置页底部新增「退出 Lingo」按钮，无需通过 Dock 或强制退出即可完全关闭应用。
- **三 Tab 面板** — 菜单栏面板重构为「翻译 / 历史 / 设置」三个 Tab。

### 变更

- 悬浮结果窗口在 3 秒后自动淡出关闭。
- 错误提示更加精确：限速、缺少 API Key、空结果、网络故障分别给出不同提示。

## [v0.1.0] — MVP 最小可用版本

> 第一个可运行的版本，仅包含核心翻译流程。

### 新增

- **菜单栏应用** — 以 `MenuBarExtra` 形式运行，不占 Dock 空间，菜单栏显示气泡图标（`character.bubble`）。
- **全局快捷键**（`⌥⌘J`）— 通过 `CGEvent` tap 注册，使用 Accessibility API（`AXSelectedText`）读取当前选中文字，立即触发翻译。
- **MyMemory 翻译** — 调用免费的 `api.mymemory.translated.net` 接口，无需注册账号或 API Key。
- **自动语言检测** — 检测 Unicode 中文字符范围（U+4E00–U+9FFF）判断翻译方向（`zh|en` 或 `en|zh`）。
- **悬浮结果窗口** — 翻译完成后，`NSPanel` 浮现在光标附近展示译文，支持文字选中和一键复制。
- **菜单栏面板** — 点击图标打开面板，支持手动输入文字并实时翻译。
- **文字转语音** — 使用 `AVSpeechSynthesizer` 朗读原文或译文。
- **自定义快捷键** — 设置弹窗支持录制新的快捷键组合。
- **辅助功能权限引导** — 首次启动时引导用户授予辅助功能权限。
- **单元测试** — 覆盖 `TranslationService`、`HotkeyManager`、`SpeechService` 的基础测试。
